### IMPORTANT: Use these exact commands in order!

## Step 1: Start Backend Server
Open PowerShell/Command Prompt in E:\CodeStory and run:

```powershell
cd E:\CodeStory
.\.venv\Scripts\Activate.ps1
uvicorn simple_backend:app --host 0.0.0.0 --port 8000 --reload
```

Wait for this message: "INFO: Uvicorn running on http://0.0.0.0:8000"
Keep this terminal open - DO NOT CLOSE IT!

## Step 2: Start Frontend (In NEW Terminal)
Open a NEW PowerShell/Command Prompt and run:

```
cd E:\CodeStory\webapp\frontend
npm install
npm start
```

Wait for: "webpack compiled successfully" 
Your browser should open http://localhost:3000 automatically

## Step 3: Test the Application ✅ NOW WORKING

## Troubleshooting

### If Backend Fails:
- Make sure you're in E:\CodeStory directory (NOT webapp subdirectory)
- Virtual environment must be activated (you should see (.venv) in prompt)
- Check if simple_backend.py exists in root directory
- The error "Could not import module simple_backend" means wrong directory

### If Frontend Fails:
- Make sure you're in E:\CodeStory\webapp\frontend directory  
- Run "npm install" first if you get module errors
- Make sure backend is running on port 8000

### If "Failed to fetch" error:
- Restart backend server
- Check http://localhost:8000 shows API message
- Make sure both servers are running simultaneously
